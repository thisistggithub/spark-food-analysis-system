基于Spark的美食数据分析可视化系统
Spark-Food-Analytics-Visualization
基于Spark的美食数据分析可视化系统，适用于大数据、计算机专业方向。

项目简介 本项目基于Spark分布式计算框架，使用PyCharm开发，在VMware搭建的Linux集群环境中运行，集成机器学习算法，实现对海量美食数据的清洗、分析和可视化展示。

技术栈 计算引擎：Spark (PySpark)

开发工具：PyCharm

集群环境：VMware Workstation（Linux虚拟机集群）

编程语言：Python 3.8+

机器学习：MLlib、Scikit-learn（聚类、分类、回归算法）

可视化：Matplotlib、Seaborn、Pyecharts

数据处理：Pandas、NumPy、Spark SQL

主要功能 数据清洗与预处理（Spark DataFrame）

探索性数据分析（美食分布/评分趋势/价格区间/菜系热度）

机器学习预测（K-Means聚类、逻辑回归分类、线性回归预测）

数据可视化大屏展示

适用场景 计算机专业课程项目

大数据方向学习实践

企业美食数据洞察与决策辅助
